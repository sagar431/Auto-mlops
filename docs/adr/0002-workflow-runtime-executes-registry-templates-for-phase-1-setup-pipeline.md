# Workflow Runtime Executes Registry Templates For Phase 1 Setup Pipeline

For Phase 1, Auto-MLOps will wire the Phase 0 **Workflow Registry** into real agent behavior by making the **Workflow Runtime** execute the registered `setup_pipeline` **Workflow Template** instead of a prompt-authored `plan_graph` skeleton. Natural-language **Workflow Selection** runs before perception, decision planning, or tool execution; the LLM may provide observations or constrained step arguments, but it may not add, remove, reorder, or skip registered setup steps. This trades open-ended agent flexibility for verified execution: missing required inputs and approval gates block before tools run, and the final workflow status is derived from the **Success Contract** rather than summarizer judgment.
